Классическая игра на развитие памяти «Memory», реализованная на языке Python. Проект отличается удобным графическим интерфейсом и расширенным функционалом, который делает игровой процесс более увлекательным и разнообразным по сравнению со стандартными версиями.

✨ Особенности игры
- Интеллектуальные челленджи: В игру интегрированы неожиданные вопросы в формате ребусов, загадок и логических задач, которые появляются в процессе игры.
- Настраиваемая сложность: Поддержка нескольких уровней сложности, позволяющих адаптировать игру под разный уровень подготовки игроков.
- Система рекордов: Встроенная таблица лучших результатов (Leaderboard) для сохранения прогресса и соревнования с другими игроками.
- Подробное руководство: Интуитивно понятные и детальные правила игры, доступные прямо в интерфейсе.

🛠 Технологический стек
Язык программирования: Python
GUI Framework: PyQt (для создания графического интерфейса)
База данных: SQLite3 (для хранения таблицы рекордов и настроек)
