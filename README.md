Проект представляет из себя известную игру "Memory", реализованную на языке python, с удобными графическим интерфейсом и с нововведениями: 
1) Неожиданные вопросы в виде ребусов, загадок или логических вопросов.
2) Разные уровни сложности.
3) Подробные правила игры.
4) Таблица лучших результатов.

Стек: Python (PyQT), SQLite3.
